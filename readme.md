### vsCode에서 실행하는 법

vsCode에서 실행시켜 보고 싶으면 파일있는 디렉토리로 이동해서 python 파일이름.py 하거나 run버튼 누르면 실행됩니다!!

실행되면 input()이 본인이 직접 입력해주는 함수여서 알고리즘 사이트에 input처럼 똑같은 순서대로 input 값 넣어주면 됩니다.
